# KrishiMitra-A-Farmers-Friend
KrishiMitra is a smart farming companion application designed to empower farmers by delivering real-time agricultural insights through data-driven technologies. The system addresses major challenges in Indian agriculture, including unpredictable weather, flood risks, inefficient irrigation practices, and lack of expert guidance.

Built with a Java-based backend, KrishiMitra integrates weather, soil, and flood APIs to
provide accurate, bilingual (Hindi–English) information through a simple and accessible
interface. The application features:
 Flood Alerts for early warnings and crop safety
 Crop Recommendations based on soil conditions and weather forecasts
 Smart Irrigation Guidance to optimize water usage
 Graphical Data Visualization for clear insights
 Farmer Reports and Analytics for informed decision-making
